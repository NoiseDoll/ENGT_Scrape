Файлы должны находиться в корне программы.

- competitors
Содержит список производителей совместимых запчастей, по строчке на каждого производителя, через tab указываются синонимы (названия с опечатками в оригинальном прайсе)

- engt.cfg
Главный конфигурационный файл, список возможных параметров:
images (boolean) - флаг для скачивания картинок
price (boolen) - флаг для создания основного прайса
resetcache (boolean) - флаг для использования кэш запчастей, создаётся автоматически после первого запуска программы.
сompetitors (boolean) - флаг для добавляения в прайс сведений о совместимых производителях (если price указан false, то в папке уже должен присутствовать ранее созданный прайс price.tsv)
phantompath (string) - путь к браузеру PhantomJS

- parts.loc
Файл локализации типов деталей. По одному типу детали на строчку, перевод отделён символом tab.
